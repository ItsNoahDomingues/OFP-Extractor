How to Convert .STF Textures to .DDS in Operation Flashpoint

Required Tools

You will need a hex editor to modify the file headers. Hex Editor Neo and HxD are recommended free tools.

Step-by-Step Conversion

  1. Open the .STF texture file in your hex editor.
  2. Remove everything from the start of the file up to DEC 2048 or HEX 00000800. The following might work: 44 44 53 20 7C 00 00 00 07 10 08 00 00 04 00 00
  3. Take the contents of a valid .DDS header file and paste it at the top of the modified texture file.
  4. Save the file as a .DDS using the format 5.6.5 (for non-alpha textures).
  5. Open the newly saved .DDS in the hex editor, remove the new DDS header added by Photoshop, then paste back the original header from the .STF file.
  6. Place the edited texture into the correct folder.

Final Step

If you are running version 1.00, 1.03, or 1.02 (patched) of the game, the texture should now appear in-game!
